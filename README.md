# Eskaptical Digital Product Uploader for WooCommerce

Bulk upload downloadable WooCommerce products using files and optional matching images.

Eskaptical Digital Product Uploader for WooCommerce is a lightweight WooCommerce bulk upload plugin designed for stores selling digital products, downloadable files, templates, ebooks, audio files, ZIP packages, and other downloadable content.

Unlike CSV import tools, this plugin allows you to upload actual downloadable files directly and automatically creates WooCommerce products for them.

---

## 🚀 Features

* Bulk upload downloadable WooCommerce products
* Automatically create virtual & downloadable WooCommerce products
* Upload digital products directly from files
* Automatic filename-based image matching
* Shared price, category, and description support
* Draft or Publish product status
* Duplicate protection
* Fast and simple workflow
* Lightweight and easy to use

## 📥 Installation

1. Upload the plugin to your WordPress website
2. Activate the plugin
3. Make sure WooCommerce is installed and active
4. Go to:

```text
WooCommerce → Bulk Uploader
```

5. Upload downloadable product files and import WooCommerce products
